# Beyond The Degree - Career Compass

**Navigating Your Journey Beyond Graduation**

A comprehensive career roadmap platform that helps students explore different career paths and educational milestones based on their chosen program (BCA).

## Features

- **Interactive Roadmaps**: Multi-phase career guidance for BCA students
- **Course Management**: Add, remove, and display course information
- **User Registration**: Simple registration system for users
- **Contact System**: Get in touch with career counselors
- **Responsive Design**: Beautiful UI with background videos and images

## Project Structure
├── router1.js # Main Express server
├── index1.html # Home page with navigation
├── registration.html # User registration form
├── contactus.html # Contact form
├── display.html # Display roadmaps
├── add.html # Add new roadmap
├── remove.html # Remove roadmap
├── aboutus1.html # About us page
├── bca_roadmap.html # BCA roadmap details
├── ConnectMongoDB.js # MongoDB connection
├── database1.js # Database operations
├── user.js # User model
├── package.json # Project dependencies


## Tech Stack

- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Frontend**: HTML, CSS, JavaScript
- **ODM**: Mongoose
- **Development**: Nodemon
